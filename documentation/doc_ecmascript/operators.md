# OPERATORS

## INCREMENT / DECREMENT

1. Pre-increment / Pre-decrement

   ```
   let i = 5;
   let x = ++i;
   console.log(i); // 6
   console.log(x); // 6
   ```

2. Post-increment / Post-decrement

   ```
   let i = 5;
   let x = i++;
   console.log(i); // 6
   console.log(x); // 5  
   ```
