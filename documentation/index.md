# JAVASCRIPT

## ECMAScript

ECMAScript adalah standar resmi untuk bahasa pemrograman JavaScript. JavaScript mengikuti aturan dan spesifikasi yang ditentukan oleh ECMAScript.

## Script

Script adalah kumpulan instruksi atau perintah yang ditulis dalam bahasa pemrograman tingkat tinggi, biasanya untuk mengotomatisasi tugas tertentu atau menjalankan logika dalam aplikasi.

## JavaScript Engine

JavaScript Engine adalah program yang bertugas membaca, menafsirkan, dan menjalankan kode JavaScript. Bisa dibilang, ini adalah otak yang membuat kode JavaScript bisa berjalan di browser atau lingkungan lain seperti Node.js.

Fungsi:

1. Membaca kode JavaScript (parsing).
2. Mengubah kode JavaScript menjadi bytecode.
3. Menjalankan kode tersebut sehingga program JavaScript bekerja.

Contoh:

1. Chrome, Edge dan Node.js menggunakan V8 Engine.
2. Firefox menggunakan SpiderMonkey.
3. Safari menggunakan JavaScriptCore.

Eksekusi:

JavaScript Engine modern menggunakan teknik Just-In-Time (JIT) compilation untuk mempercepat eksekusi kode. Jadi, meskipun JavaScript awalnya interpreted, engine mengubahnya menjadi lebih cepat saat dijalankan.
